# PreserFlo MicroShunt Surgery Risk Calculator

A web-based tool for estimating 5-year surgical failure risk after PreserFlo MicroShunt glaucoma surgery based on preoperative patient characteristics.

## Purpose

This calculator implements a Cox Proportional Hazards regression model to predict the probability of surgical failure within 60 months following PreserFlo MicroShunt surgery.

**IMPORTANT:** This is a preliminary model for research and educational purposes only. It should NOT be solely relied upon for clinical decision-making.

**Creator:** Javad Ghanbarnia (MD)

## Quick Start

Open the HTML file in any web browser. No installation required.
